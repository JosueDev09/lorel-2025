# Videos para el Hero Slider

Coloca aquí los videos para el slider del hero.

## Video requerido:

**lorel-promo.mp4** - Video promocional de LOREL
- Formato: MP4 (H.264)
- Dimensiones recomendadas: 1920x1080px
- Duración recomendada: 10-30 segundos
- Peso recomendado: Menos de 5MB (para carga rápida)

## Consejos:
- Optimiza el video para web usando compresión H.264
- Mantén un aspecto ratio de 16:9
- El video se reproducirá en loop automáticamente
- El video será muted (sin sonido) por defecto para mejor UX
- Considera crear versiones responsive (más ligeras para móviles)

## Herramientas de optimización:
- HandBrake (gratuito)
- Adobe Media Encoder
- FFmpeg (línea de comandos)
- CloudConvert (online)
