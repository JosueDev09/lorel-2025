# Imágenes para el Hero Slider

Coloca aquí las imágenes para el slider del hero. Se recomienda:

## Imágenes requeridas:

1. **slide-1.jpg** - Primera imagen del slider
   - Dimensiones recomendadas: 1920x1080px
   - Formato: JPG o PNG
   - Sugerencia: Productos de cintas y etiquetas LOREL

2. **slide-2.jpg** - Segunda imagen del slider
   - Dimensiones recomendadas: 1920x1080px
   - Formato: JPG o PNG
   - Sugerencia: Proceso de producción o instalaciones

3. **video-poster.jpg** - Imagen de portada para el video
   - Dimensiones recomendadas: 1920x1080px
   - Formato: JPG o PNG
   - Esta imagen se muestra mientras carga el video

## Consejos:
- Usa imágenes de alta calidad y resolución
- Mantén un aspecto ratio de 16:9 para mejor visualización
- Optimiza las imágenes para web (compresión sin pérdida de calidad)
- Los colores corporativos (#1F3C68, #9C2D32, #231F20) se verán bien con el overlay
